信息比对助手（Linux 版）— 统信 UOS 使用说明
========================================

一、在 Windows 上打 Linux 包（给开发者）
----------------------------------------
1. 安装 .NET 8 SDK。
2. 在仓库根目录下，运行：
   - PowerShell：  .\packaging\uos\publish-linux.ps1
3. 或在项目根执行：
   dotnet publish InfoCompareAssistant\InfoCompareAssistant.csproj -c Release -r linux-x64 --self-contained true -o packaging\uos\build

生成目录 packaging\uos\build，可将整个文件夹打成 zip/tar 拷到 UOS 机器。

二、在统信 UOS 上安装（给最终用户）
----------------------------------
1. 将发布得到的文件夹（内含名为 InfoCompareAssistant 的无扩展名主程序、大量 .dll/.so 等）拷到 UOS 上，例如家目录下，解压到任意路径（路径中建议少用空格）。

2. 依赖（图形窗口需要 WebKitGTK 等，版本随系统可能不同）：
   sudo apt update
   sudo apt install -y libgtk-3-0
   按系统仓库搜索并安装与 Photino 匹配的 webkit 之一，例如：
   sudo apt install -y libwebkit2gtk-4.0-37
   若该包不存在，可试：
   sudo apt install -y libwebkit2gtk-4.1-0

3. 推荐：一键安装（与安装包在同一文件夹，勿单独挪走 .desktop 文件）
   在文件管理器中找到本目录下的：InfoCompareAssistant-Install.desktop（在界面里显示名可能为「一键安装」）双击。
   - 若出现「未信任/无法启动」：右键 → 属性/权限/信任（不同桌面文案略有不同），或改为在终端中执行步骤 4。
   - 成功时一般会弹出系统自带的「提权/授权」窗口，输入本机账户密码，安装到 /opt/InfoCompareAssistant 并写开始菜单。

4. 命令行方式（与一键安装等效，便于看错误信息）：
   cd /你的安装包所在路径
   chmod +x install.sh install-to-opt.sh InfoCompareAssistant 2>/dev/null; true
   sudo ./install.sh
   也可使用旧入口：sudo ./install-to-opt.sh
5. 主程序在终端也可直接运行（已安装到 /opt 时）：
   /opt/InfoCompareAssistant/InfoCompareAssistant
   或在未安装、仍在安装包目录时（便于看报错）：
   chmod +x ./InfoCompareAssistant; ./InfoCompareAssistant
6. 若 Photino 窗口因依赖原因无法打开，可改用系统浏览器方式访问本机 Kestrel（在安装包目录下）：
   ./InfoCompareAssistant --browser
   再按终端提示的 http://127.0.0.1:端口 在浏览器中打开。

三、数据与备份
--------------
- 数据目录：一般为 ~/.local/share/InfoCompareAssistant/（XDG 本地数据），内含 SQLite 数据库 app.db。
- 退出时自动备份到：同一数据根目录下的「数据备份」子文件夹（不再使用 D:\ 盘，除非是在 Windows 且 D 盘存在时）。

四、CPU 架构
------------
- 以上默认脚本为 linux-x64（常见 PC/服务器）。
- 若目标为 arm64 设备，发布时使用：-r linux-arm64
  并自行复制本说明中的依赖安装命令在对应架构上试装。
